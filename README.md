# ⚽ Football Game

A desktop football game developed in **Unity** using **C#**. This project was built to explore the fundamentals of Unity game development, including physics-based movement, collision detection, user input handling, and UI integration.

---

## 📖 About the Project

This project is a simple football game where players control the football using keyboard and mouse inputs. The game uses Unity's physics engine to simulate realistic ball movement. When the ball enters the goal, the game detects it, displays a goal message, and automatically resets the ball to continue playing.

The project helped me gain hands-on experience with Unity's game engine, scripting, and gameplay mechanics.

---

## ✨ Features

* ⚽ Physics-based football movement using Unity's Rigidbody
* 🎮 Keyboard and mouse controls
* 🥅 Goal detection using trigger colliders
* 🏆 Goal notification using Unity UI
* 🔄 Automatic ball reset after scoring
* 🎯 Interactive gameplay mechanics

## 🚀 Getting Started

### Prerequisites

* Unity Hub
* Unity Editor(2022.3.62f1):(recommended: use the same Unity version in which the project was created)

### Installation

1. Clone the repository.

```bash
git clone https://github.com/<your-username>/Football-Game.git
```

2. Open the project using Unity Hub.

3. Open the main scene from the **Assets/Scenes** folder.

4. Click **Play** to start the game.
